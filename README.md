#DESCRIPTION
The objective of this task is to build a machine learning model for predictive analysis using PySpark MLlib. The dataset is analyzed and relevant features are selected to train a Linear Regression model for predicting sales outcomes. The task includes feature selection, feature engineering, model training, prediction generation, and model evaluation using RMSE. This project demonstrates the implementation of machine learning techniques for predictive analytics on large datasets using PySpark.
